# Portfolio

This is the repository for my personal portfolio website.

## Tech Stack

- Nuxt
- Tailwind CSS
- GSAP

## Setup

Install dependencies:

```bash
# pnpm
pnpm install

# npm
npm install

# yarn
yarn install
```

# Development

Start the development server on http://localhost:3000

```bash
# pnpm
pnpm run dev

# npm
npm run dev

# yarn
yarn dev
```

# Production

Build the application for production:

```bash
# pnpm
pnpm run build

# npm
npm run build

# yarn
yarn build
```
